# Whole World

Paint on a global map. That's it. No login, no nonsense. You get a random ID (gestXXXX) and you're in.

## What you can do

- Paint pixels on a world map in real-time
- Pick from 50+ colors
- Erase stuff if you mess up
- Change the color of pixels you already painted
- See who painted the most on the leaderboard
- Search for locations to paint there
- Your paintings stick around (stored in JSON)

## How to run it

1. Clone this
```bash
git clone https://github.com/joelmohh/wholeworld.git
cd wholeworld
```

2. Install stuff
```bash
npm install
```

3. Start
```bash
node app.js
```

Go to http://localhost:3000

## How to use

**Painting**: Click Paint, pick a color, click squares. Hold space + drag to paint continuously.

**Changing colors**: Just click on a pixel you already painted and select a new color.

**Erasing**: Click Paint, then click Erase, then click the pixels you want gone.

**Leaderboard**: Click the star icon to see who painted the most.

**Search**: Click the map icon, type a place name, and jump to it.
